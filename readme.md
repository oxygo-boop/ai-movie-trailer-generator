## 🌐 Web App (Streamlit)

This project also includes a Streamlit web app interface.

### Run in Google Colab

1. Open the notebook
2. Run all cells
3. Click the ngrok link that appears
4. Use the web interface to generate your trailer

### Features of the Web App

- Clean dark cinematic UI
- Sidebar inputs for movie details
- Drag and drop video upload
- Live story and script preview
- Audio player for voice-over
- Video player for final trailer
- One click download button
