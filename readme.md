# 🎬 AI Movie Trailer Generator

An AI-powered cinematic trailer generator built with:

- Google Gemini
- Edge TTS
- MoviePy
- OpenCV
- Python

---

## 🚀 Features

✅ Generate a movie story  
✅ Create a 3+ scene trailer script  
✅ Extract cinematic narration  
✅ Convert narration to AI voice-over  
✅ Upload custom scene videos  
✅ Auto-assemble final trailer with effects

## 🛠 Tech Stack

- Python
- Google Gemini API
- Edge TTS
- MoviePy
